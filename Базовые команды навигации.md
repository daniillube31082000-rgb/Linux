В терминале есть несколько основных команд, которые позволяют вам свободно гулять по директории и видеть, что там лежит. Давайте знакомиться

### **pwd**
+ **От англ. print working directory.** Показывает, в каком каталоге вы сейчас находитесь.
+ Для системы это актуально, потому что многие команды работают с вашими текущими путями.
+ Например:  

  ```pwd```

                  
Если вы находитесь в домашней папке пользователя, вывод может быть /home/username или что-то подобное.  
### **ls** 

+ От англ. list — **список.** Показывает список файлов в текущем (или указанном) каталоге.
+ Простой пример:  
```ls```

                  
Вы увидите имена файлов и папок. Если каталог пуст, ничего не будет отображено.
+ **Варианты:**

+ ``ls -l`` — детальный список с датой изменения, правами, владельцем и размером.
+ ``ls -a`` — показать скрытые файлы, начинающиеся с точки (например, .bashrc).
+ ``ls -la`` — можно комбинировать опции (l + a).  
### **cd**

+ **Change directory** — переход в другой каталог.
+ **Примеры:**

+ ``cd /`` — переход к корню файловой системы (самая вершина).
+ ``cd ..`` — подняться на один уровень вверх (как бы обратно).
+ ``cd ~`` — перейти в домашнюю папку текущего пользователя (~/ равносильно /home/username).
+ ``cd /home/user/Documents`` — перейти по абсолютному пути.
+ ``cd Documents`` — перейти по относительному пути (если вы уже в /home/user).
+ Чтобы вернуться в директорию, где вы вы только что были — наберите ```cd -```. Это удобно, если нужно прыгать взад-вперед.  

### **Навигация: абсолютные и относительные пути**  

+ **Абсолютный путь всегда начинается** ``с / (корня)``.  

+ **Например** , ```/home/user/Documents```. Он полный: для перехода не важно, где вы сейчас.  

+ **Относительный путь** задаётся относительно вашей текущей директории.  

+ Например, если вы уже в ``/home/user``, достаточно ``cd Documents``.
### **Имена, начинающиеся с точки**
+ В Linux файлы, начинающиеся с точки ``(.)``, считаются скрытыми.
+ ``.bashrc``, ``.gitignore``, ``.config``, и т. д.  

+ Чтобы увидеть их, используйте ``ls -a``.
+ Символы ``.``и ``..``— специальные каталоги. ``.`` означает текущую директорию, ``..`` означает родительскую директорию. Символ ``~`` указывает на домашний каталог пользователя. Его часто используют как сокращение для ``/home/username``.
Команда ``mkdir`` **(make directory)** создаёт новый каталог, но не используется для перемещения по файловой системе. Для навигации применяются только ``pwd, ls и cd``.
### **Итог**
+ ``pwd`` — показывает, где вы находитесь
+ ``ls`` — выводит содержимое каталога
+ ``cd`` — позволяет перемещаться по директориям
+ **Абсолютные пути** начинаются с ``/``, а **относительные** зависят от текущей позиции  

Уверенное владение этими командами — первый шаг к тому, чтобы свободно ориентироваться в Linux. В следующем уроке мы потренируемся применять их на практике и научимся работать быстрее с помощью некоторых приёмов.
